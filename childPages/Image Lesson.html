Steps in building an Image

  • Pick an Image (carefull of the file extension in the URL)
  • Record URL, cite it, comment for date access, and record origonal dimesions
  • Use blockquote and comment as appropriate
  • Use img-tag with atributes src, alt, title, and style with width-height
